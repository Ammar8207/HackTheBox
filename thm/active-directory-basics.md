# THM - Active Directory Basics
**Platform:** TryHackMe

## What It Covered
Introduction to Microsoft Active Directory — how 
organizations manage users, devices, and access control.

## What I Learned
- AD is used in almost every corporate environment 
  to manage multiple systems centrally
- Domain Controllers handle authentication and 
  authorization across the network
- AD misconfigurations are one of the most common 
  attack vectors in real-world breaches
